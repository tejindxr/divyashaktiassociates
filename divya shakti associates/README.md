# Divyashakti Associates WordPress theme

This is a standalone classic WordPress theme created from the content in `dsa` and the visual direction in `dsa/stitch design`.

## Install

Copy the `divya shakti associates` folder into `wp-content/themes/`, activate **Divyashakti Associates** in WordPress, and optionally assign a menu to the **Primary Menu** location. The fallback navigation works without a menu.

The enquiry form submits through WordPress `admin-post.php` and sends to the WordPress admin email using `wp_mail`. Configure SMTP on the WordPress install for reliable delivery.
